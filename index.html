Drawer Translate

Open-source cross-platform desktop application to translate text using different machine translation APIs. It has been developed under the idea of it doesn't interfer with the rest of our work, so the app is a window at the bottom of your screen that can be resized, minimized, hidden or auto-hidden.

On Linux

Just download the self-contained AppImage executable for Linux and double-click it to run. Don't forget to set execution permission to the downloaded file before try to execute it.

Once it is running, go to Settings and select you prefered Translate API and the desired source and target translation languages. In addition, some others settings can be configured, like translate text on mouse selection and/or on mouse copy, auto-hide this app or the appearance of it.

On Windows

Simply download the Installation Wizard executable for Windows and double-click it to run. Select the location where you want to install Drawer Translate and it automatically will create a shortcut in the Windows Start Menu.

Once it is installed, go to the start menu, run Drawer Translate and it is ready to start translating. In the Settings menu you can select a different Translate API if you wish and change the source and target translation languages. In addition, some others settings can be configured, like translate text on mouse selection and/or on mouse copy, auto-hide this app or the appearance of it.
